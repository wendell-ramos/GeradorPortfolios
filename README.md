# Gerador de Portfolios

Aplicacao web para criar portfolios interativos de desenvolvedores por meio de um editor guiado e personalizavel.

## Recursos atuais

- Dados de exemplo ou preenchimento manual
- Identidade profissional, experiencias, stack, projetos e contatos
- Secoes configuraveis e personalizadas
- Upload de foto de perfil e capas de projetos
- Cores globais e personalizacao visual por area
- Templates Desktop retro, Terminal hacker e Docs moderno
- Visualizacao completa e responsiva do site gerado

## Tecnologias

- React
- TypeScript
- Vite
- CSS
- Oxlint

## Executar localmente

```bash
npm install
npm run dev
```

Acesse `http://127.0.0.1:5173/` no navegador.

## Validacao

```bash
npm run lint
npm run build
```
